# MLmodels
A collection of Machine Learning projects covering data preprocessing, feature engineering, model training and evaluation. Built using Python, TensorFlow, Scikit-learn and Pandas.

## Projects

### 1. Heart Disease Prediction
- **Dataset:** UCI Heart Disease Dataset (918 patients)
- **Type:** Binary Classification
- **Model:** Neural Network with L2 Regularization
- **Accuracy:** 91.3%
- **Libraries:** TensorFlow, Scikit-learn, Pandas, NumPy

## Tech Stack
- Python
- TensorFlow
- Scikit-learn
- Pandas
- NumPy
- Matplotlib

## How to Run
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Open Jupyter Notebook
4. Run the notebook cell by cell
